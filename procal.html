Installation Steps
⚠ K-Line Driver Installation Is the Hard Part

The K-Line USB driver is required for ProCal to communicate with the ECU. Windows often blocks unsigned drivers. Read and watch the full installation video before starting — the end of the video contains critical tips.

1
Download and Extract the Software

Download the ProCal software file above. Extract it using WinZip or WinRAR (free at winrar.com). Do not run the installer directly from inside the ZIP/RAR.

2
Disable Driver Signature Enforcement

Before installing the K-Line driver, you must temporarily disable Windows driver signature enforcement. This is required because the K-Line driver is not digitally signed.

⚠ This only disables enforcement for the current session — it re-enables automatically on the next reboot.
  1. Click the Start button, then the Power icon.
  2. Hold SHIFT and click Restart.
  3. On the blue menu: Troubleshoot → Advanced Options → Startup Settings → Restart.
  4. After restart, press 7 or F7 to select "Disable driver signature enforcement."
  5. Windows will boot normally — install the K-Line driver now before rebooting.
3
Install the K-Line Driver

Inside the extracted ProCal folder, find the K-Line driver installer and run it. Complete the installation while driver signature enforcement is still disabled (from Step 2).

Do NOT reboot between Step 2 and Step 3 — rebooting will re-enable signature enforcement and the driver install will fail.
4
Install ProCal Software

Run the ProCal installer from the extracted folder and follow the on-screen instructions to complete installation.

5
Check the COM Port Number

ProCal requires the USB device to be on COM7 or lower. To check and change it:

  1. Plug in the FiTech USB/handheld cable.
  2. Press Windows Key + X and open Device Manager.
  3. Expand Ports (COM & LPT) and find the FiTech/Prolific device.
  4. If the COM number is 8 or higher, right-click → Properties → Port Settings → Advanced, and change it to COM1–COM7.
  5. Unplug and replug the device to confirm.
6
Connect Handheld and Launch ProCal

Plug your 2-Dot handheld into the laptop with your USB-C cable. On the handheld, scroll to the bottom of the Main Menu → Handheld Software Mode Selection → Laptop Software Connection. Then launch ProCal on your laptop.

Troubleshooting & Windows 11
ℹ Windows 11 — Extra Steps Required

ProCal is older 16-bit software. Windows 11 (64-bit) requires an additional optional feature to be installed before the setup program will run. See the accordion below for detailed steps.

Windows 11 — How to Run the ProCal Installer
Windows 11
›

Windows 11 (64-bit) cannot run the older 16-bit ProCal installer unless you install a special Microsoft feature first. Here's how:

  1. Click the Start button.
  2. Type Installed apps and press Enter.
  3. Click Optional features.
  4. Click Add a feature at the top.
  5. Search for "16-bit InstallShield Support".
  6. Check the box beside it and click Install.
  7. Wait for the install to finish, then try running the ProCal installer again.
  8. If prompted to reboot, click Restart.

After this step, ProCal should install and run on Windows 11. You will still need to complete the K-Line driver steps (disable driver signature enforcement, etc.).

Disabling Driver Signature Enforcement — Detailed Steps
Win 10 & 11
›

Temporary method (recommended — resets on reboot):

  1. Click Start → Power icon.
  2. Hold SHIFT and click Restart.
  3. Blue screen appears: click Troubleshoot.
  4. Click Advanced Options.
  5. Click Startup Settings, then Restart.
  6. After restart, press 7 or F7: "Disable driver signature enforcement."
  7. Windows boots normally — install your K-Line driver now.
  8. Restart when done — enforcement re-enables automatically.

Permanent method — Test Mode (advanced users only):

Open Command Prompt as Administrator and run:

bcdedit /set testsigning on

Restart your computer. Windows will show a "Test Mode" watermark. To turn it off later:

bcdedit /set testsigning off

⚠ Test Mode is a security risk and not recommended for general use. Only use if the temporary method is not practical.

ProCal Won't Connect — COM Port Fix
Troubleshooting
›

ProCal only works on COM ports 1 through 7. If your device is assigned COM8 or higher, ProCal won't see it. Here's how to fix it:

  1. Plug in the FiTech USB device.
  2. Press Windows Key + X → Device Manager.
  3. Expand Ports (COM & LPT).
  4. Look for "Prolific USB-to-Serial Comm Port" or "USB Serial Port (COMx)".
  5. If the COM number is 8 or higher: right-click → Properties → Port Settings → Advanced.
  6. In the COM Port Number dropdown, select a port from COM1 to COM7.
  7. Click OK → OK, then unplug and replug the device.
  8. Check Device Manager again to confirm the new COM number.

If all low COM ports are "in use", you can still assign to one — Windows may warn you, but as long as nothing else is actively using that port, it will work.
