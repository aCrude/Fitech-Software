⚠ 32-Bit LS ECU Identification

New 32-bit LS ECUs began shipping for new systems and replacement/warranty cases after August 20, 2025.

How to identify: 32-bit ECUs have a date code and serial number laser etched on the main connector. See TSB #0006 for detailed identification photos.

Compatibility: The 32-bit ECU requires a 2-Dot handheld running software v1.79 or newer. It is NOT compatible with older handhelds or older software. Using old software with a 32-bit ECU will result in wrong dashboard readings and "ECU No Response" errors.

If you are unsure which ECU you have, contact FiTech support at (951) 340-2624 before proceeding.
